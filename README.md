유런 6기 여름방학 프로젝트

[ISIC 2024 피부암 탐지 챌린지](https://www.kaggle.com/competitions/isic-2024-challenge/overview)

- 파인튜닝된 모델을 학습한 후, 정확도, 재현율, 정밀도 등의 지표를 사용해 모델 성능을 평가
